# Projects

This repository is a collection of my personal and academic projects, spanning multiple domains such as computer vision, machine learning, backend development, algorithms, and experimental ideas.

Each project lives in its own folder and is mostly self-contained, with its own code, documentation, and dependencies where applicable.

The purpose of this repository is:
- to track my learning and experimentation over time
- to organize different ideas and prototypes in one place
- to serve as a reference for implementations, concepts, and pipelines
- to showcase practical work rather than isolated snippets

## Structure

The repository follows a simple structure:

projects/
│
├── project-name-1/
│ ├── README.md
│ ├── src/
│ └── ...
│
├── project-name-2/
│ ├── README.md
│ └── ...
│
└── ...


Each project folder may include:
- source code
- datasets or dataset links
- model files (when appropriate)
- a dedicated README explaining the project

## Notes

- Some projects are prototypes or experiments and may evolve over time.
- Not all projects are production-ready; many are learning-focused.
- Large files such as trained models or videos may be excluded using `.gitignore`.
